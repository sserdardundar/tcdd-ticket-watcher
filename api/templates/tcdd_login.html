Option 2: Manual Token Fallback (Guaranteed to work)

  1. Open TCDD e-Bilet in a new tab.
  2. Login with your TCDD account (so the User Token is generated).
  3. Press F12 to open Developer Tools, go to the Network tab.
  4. Search for a train trip. Click any API request (like train-availability).
  5. Look at the Request Headers. Copy the values for Authorization and User-Authorization.
